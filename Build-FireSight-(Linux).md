
### Debian, Ubuntu

Linux installation is easy. The following has been tested on :
* Raspbian (Raspberry Pi)
* Ubuntu 12.04 LTS (Chromium Pixel)

<pre>
sudo apt-get install git
git clone git://github.com/firepick1/FireSight
cd FireSight
sudo ./build
</pre>
